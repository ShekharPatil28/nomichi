# Nomichi

Nomichi is a travel community and trip management platform built with Next.js, TypeScript, Supabase, and Vercel.

## Features

* User trip enquiry form
* Lead capture and management
* Trip creation and management
* Admin dashboard
* User authentication
* Local development setup with environment variables

## Tech Stack

* Next.js
* TypeScript
* Supabase
* Vercel
